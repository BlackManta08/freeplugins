thread (30fps)\n- Atomic variables for thread-safe communication\n- Parameter changes queued via JUCE system\n\n### DSP Signal Flow\n```\nAudio Input\n    ↓\n[Pitch Detection] → Detected Frequency\n    ↓\n[Scale Quantization] → Target Frequency\n    ↓\n[Vibrato LFO] → Modulated Frequency\n    ↓\n[Pitch Shifter] (SoundTouch)\n    ↓\n[Formant Adjustment]\n    ↓\nAudio Output\n```\n\n### Parameters (Thread-Safe Atomics)\n- `detectedPitch` – Real-time pitch from detector\n- `targetPitch` – Quantized/processed pitch\n- `manualPitchMode` – Piano roll override active\n- `manualTargetPitch` – User-set frequency\n\n---\n\n## 📚 RESOURCES\n\n- **JUCE Docs**: https://docs.juce.com\n- **SoundTouch**: https://soundtouch.surina.net\n- **Yin Algorithm**:
