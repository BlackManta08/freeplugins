# FreeAutotune - Free Autotune Plugin for FL Studio\n\nDown with expensive plugins! FreeAutotune is a professional-grade pitch correction plugin with an intuitive visual editor, comparable to industry-standard autotune.\n\n## Features\n\n### Core Pitch Correction\n- ✅ **Real-time Pitch Detection** – Yin algorithm with accuracy tuning\n- ✅ **Scale Quantization** – Major, Minor, or Chromatic scales\n- ✅ **Smooth Pitch Shifting** – SoundTouch library for high-quality results\n- ✅ **Correction Amount Control** – Blend between original and corrected signal\n\n### Advanced Effects\n- 🎵 **Vibrato** – Adjustable amount (0-100%) and rate (0.5-10 Hz)\n- 📐 **Formant Preservation** – Maintain vocal character with ±12 semitone shifts\n- 🎹 **Visual Piano Roll Editor** – Real-time pitch curve editing\n- ✏️ **Manual Mode** – Click and drag to override automatic detection\n\n### User Interface\n- 📑 **Two-Tab Design** – Controls & Piano Roll views\n- 🎨 **Real-time Visualization** – Detected vs. target pitch display\n- 🎚️ **Parameter Labels** – Clear control identification\n- 🔄 **Live Audio Feedback** – See changes in real-time\n\n## Building\n\n### Prerequisites\n- **CMake** 3.15+\n- **C++17** compatible compiler (GCC 7+, Clang 6+, MSVC 2017+)\n- **JUCE** (auto-downloaded via CMake)\n- **SoundTouch** library\n\n### macOS (ARM64/Intel)\n```bash\ngit clone https://github.com/BlackManta08/freeplugins.git\ncd freeplugins\ngit submodule add https://github.com/juce-framework/JUCE.git\nmkdir build && cd build\ncmake -GXcode ..\ncmake --build . --config Release\n```\n\n### Windows (Visual Studio)\n```bash\ngit clone https://github.com/BlackManta08/freeplugins.git\ncd freeplugins\ngit submodule add https://github.com/juce-framework/JUCE.git\nmkdir build && cd build\ncmake -G \"Visual Studio 16 2019\" ..\ncmake --build . --config Release\n```\n\n### Linux (GCC)\n```bash\ngit clone https://github.com/BlackManta08/freeplugins.git\ncd freeplugins\ngit submodule add https://github.com/juce-framework/JUCE.git\nmkdir build && cd build\ncmake ..\ncmake --build . --config Release\n```\n\n## Plugin Output Locations\n\nAfter building, your plugin will be in:\n- **macOS**: `build/FreeAutotune_artefacts/Release/AU/` and `build/FreeAutotune_artefacts/Release/VST3/`\n- **Windows**: `build\\FreeAutotune_artefacts\\Release\\VST3\\`\n- **Linux**: `build/FreeAutotune_artefacts/Release/VST3/`\n\nCopy to your DAW's plugin folder:\n- **FL Studio** (Windows): `C:\\Program Files\\Image-Line\\FL Studio 20\\Plugins\\VST3\\`\n- **macOS AU**: `~/Library/Audio/Plug-Ins/Components/`\n- **macOS VST3**: `~/Library/Audio/Plug-Ins/VST3/`\n\n## Usage Guide\n\n### Basic Workflow\n1. Insert FreeAutotune on a vocal track\n2. Adjust \"Pitch Correction\" slider (0-100%)\n3. Select desired scale (Major/Minor/Chromatic)\n4. Play audio and listen for corrections\n\n### Advanced Techniques\n\n**Piano Roll Mode:**\n- Switch to \"Piano Roll\" tab\n- Click and drag on the grid to manually set target pitches\n- Blue line = detected pitch, Orange bar = target pitch\n- Release mouse to snap to grid\n\n**Vibrato Effect:**\n- Increase \"Vibrato Amount\" for subtle oscillation\n- Adjust \"Vibrato Rate (Hz)\" for speed (typical: 4-6 Hz)\n- Works on both auto-corrected and manual-mode pitches\n\n**Formant Preservation:**\n- Use \"Formant Shift\" to maintain vocal character when shifting large intervals\n- Positive values brighten, negative values darken vocal tone\n- Typically ±3-6 semitones for natural results\n\n## Parameters\n\n| Parameter | Range | Default | Description |\n|-----------|-------|---------|-------------|\n| Pitch Correction | 0-100% | 80% | Amount of automatic pitch correction |\n| Scale | Major/Minor/Chromatic | Major | Target scale for quantization |\n| Vibrato Amount | 0-100% | 0% | Intensity of vibrato modulation |\n| Vibrato Rate | 0.5-10 Hz | 5 Hz | LFO frequency for vibrato |\n| Formant Shift | -12 to +12 st | 0 | Formant preservation adjustment |\n| Bypass | On/Off | Off | Disable processing |\n\n## Architecture\n\n### Core Modules\n- **PitchDetector** – Yin algorithm implementation + scale quantization\n- **PitchShifter** – SoundTouch wrapper for high-quality pitch shifting\n- **PluginProcessor** – Main DSP engine and parameter management\n- **PluginEditor** – JUCE-based UI with tabs and controls\n- **PianoRollComponent** – Visual pitch editing interface\n\n### Supported Formats\n- ✅ VST3 (Windows, macOS, Linux)\n- ✅ AU (macOS)\n- 🔄 Standalone (planned)\n\n## Known Limitations\n\n- Single-channel processing (stereo sums to mono internally)\n- ~30ms latency (tuned for real-time performance)\n- Vibrato LFO affects overall pitch (not just oscillation)\n- Formant shift is approximate (not full spectral analysis)\n\n## Performance\n\n- CPU Usage: ~3-5% per instance (i7-9700K, 44.1kHz)\n- Memory: ~5-10 MB per instance\n- Latency: ~30ms (configurable in future versions)\n\n##
